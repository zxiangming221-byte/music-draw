## 🎨 介绍

Music Draw 是一款交互式音乐发现工具。基于弹簧物理的 3D 卡片旋转动画，配上 Google Rainbow 渐变色彩和毛玻璃 UI，带来沉浸式的音乐探索体验。

### ✨ 特色

- 🎯 **3D 卡片旋转** — 真实弹簧物理动效
- 🌈 **Google Rainbow 配色** — 六色渐变过渡
- 🪟 **毛玻璃 UI** — Frosted glass 设计语言
- 📱 **跨平台** — 浏览器打开即用，也可打包为 Android APK

### 🛠 技术

- 纯 HTML5 / CSS / JavaScript（单文件）
- CSS 3D Transforms + Spring Physics
- Capacitor 打包为 Android APK

## 📥 使用

```bash
# 浏览器直接打开
open index.html

# 或用 Capacitor 打包 APK
npx cap add android
npx cap sync
```
